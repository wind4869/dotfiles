dotfiles
========

Backups of my configuration files.

config files
--------
- vim: `.vimrc`  
- git: `.gitconfig`  
- hexo: `_config.yml`

manual install
--------
    ln -s ~/repos/dotfiles/vim/.vimrc ~/.vimrc
    ln -s ~/repos/dotfiles/git/.gitconfig ~/.gitconfig
    ln -s ~/repos/dotfiles/hexo/_config.yml ~/hexo/_config.yml
